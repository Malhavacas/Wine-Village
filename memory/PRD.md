# Wine Village - PRD

## Problem Statement
Build a landing page website for "Wine Village" - Portuguese Petiscaria (tapas restaurant) in Águeda, Portugal. Mobile-first, bilingual PT/EN, no database.

## User Personas
- **Local customers** searching on mobile for nearby restaurants
- **Tourists** visiting Águeda looking for authentic Portuguese food
- **Group organizers** looking for dinner venue

## Core Requirements
- Hero section with CTA (reserva, menu)
- About section with restaurant story
- Menu grid organized by category tabs (Entradas, Pratos, Risotos, Sobremesas, Bebidas)
- Customer reviews showcase (4.5 stars, 273 reviews)
- Location with embedded Google Maps
- Click-to-call and social media contact
- Language toggle PT/EN
- Download HTML button
- Mobile-first responsive design

## What's Been Implemented (Dec 2025)
- Full React landing page with all sections
- Bilingual support (PT-PT / EN) with context-based toggle
- Sticky header with phone CTA and language switch
- Menu tabs with accurate food images for each dish
- Reviews section with Google rating badge
- Location section with Google Maps embed
- Footer with opening hours and download button
- Smooth scroll animations with Intersection Observer
- SEO meta tags and Open Graph

## User Feedback Applied
- Changed hero background from wine cellar to restaurant interior
- Replaced all restaurant-interior images with food-only closeups
- Ensured each dish image accurately represents the actual dish
- Major redesign: dark editorial style with organic shapes, pill buttons, italic gold emphasis, glass-morphism cards (based on user reference screenshots)
